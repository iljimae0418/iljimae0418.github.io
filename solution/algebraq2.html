From Sophie Germain identity, note that \(4x^4+1 = ((x-1)^2 + x^2)(x^2 + (x+1)^2) \). Then we can simplify the expression in the problem to $$\frac{((0^2+1^2)(1^2+2^2))((2^2+3^2)(3^2+4^2))...((10^2+11^2)(10^2+11^2))}{((1^2+2^2)(2^2+3^2))((3^2+4^2)(4^2+5^2))...((11^2+12^2)(12^2+13^2))} = \frac{1}{12^2+13^2} = \frac{1}{133}$$, so \(m+n = 313\).
