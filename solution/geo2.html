This problem can be solved with a direct application of stewart's theorem. The given condition is equivalent to \(AP(BP+CP) = 5BP = 3CP\) and from this we get three relationships:
  • \(AP = \frac{3CP}{BP+CP}\) ... (1)

  • \(AP = \frac{5BP}{BP+CP}\) ... (2)

  • \(BP = \frac{3CP}{5}\) ... (3)
Substituting (3) in (1) or (2), we get \(AP = \frac{15}{8}\). Let \(CP = x\), then by Stewart's Theorem, $$ 24x = \frac{8x}{5}(\frac{225}{64} + \frac{3x^2}{5}) \rightarrow x = 0, \pm \frac{35}{8} $$ but \(x > 0\), so \(x = \frac{35}{8}\). Thus $$ BC^2 = \frac{64x^2}{25} = 49 $$.
